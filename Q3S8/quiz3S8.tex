\setcounter{required}{7}
\begin{required} 
Consider the following graph $G(V, E, w)$. You should:
\begin{itemize}
\item Clearly indicate the order in which \textbf{Prim's algorithm} adds the edges to the minimum-weight spanning tree, \textbf{using $A$ as it's starting vertex}.
\item You may simply list the order of the edges; it is not necessary to exhibit the state of the algorithm (i.e., the disjoint-set data structure) at each iteration.
\end{itemize}

\begin{proof}[Answer]

\begin{center}
After performing Prim's algorithm on the following graph of $G$, we will be left with the Minimum Weight Spanning Tree of $G$:\\
Here is the order in which Prim's algorithm will add the edges to the Minimum Weight Spanning Tree of $G$:\\
$[\{A, C\}, \{A, B\}, \{B, D\}, \{D, E\}, \{E, F\}] $\\
\end{center}
\begin{center}
Prim's algorithm will not add the edges of [\{C, E\}, \{C, B\}, \{D, F\}] because it will create a cycle within the Minimum Weight Spanning Tree of $G$. \\
\end{center}
